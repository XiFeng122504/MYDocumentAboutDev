# Personal Documentation Repository

本仓库用于整理与记录个人在 C++、Qt 以及 Unity3D 等方向的学习与开发过程。内容涵盖服务端、客户端、图形界面与游戏开发等多个领域，旨在形成一个持续更新、可复用的知识与文档中心。

---

## 📁 仓库结构

### 1. cppserver/
记录基于 C++ 实现的服务端相关内容，包括但不限于：
- 服务器架构设计思路
- 网络通信模型（如 Reactor、Proactor、IOCP、epoll 等）
- 线程池、连接池、定时器等基础组件
- 项目实践过程中的总结与问题记录

### 2. cppclient/
主要包含 C++ 客户端相关的开发文档，例如：
- 客户端通信模块设计
- 协议解析、序列化/反序列化
- 跨平台相关笔记
- 架构、模块划分与实现细节记录

### 3. qt/
用于记录 Qt 框架的学习与使用：
- Qt Widgets / Qt Quick (QML) 的开发笔记
- 信号与槽机制、界面布局、资源管理等
- 实际项目中遇到的问题与解决方案
- 常用组件示例与总结

### 4. unity3d/
整理 Unity3D 游戏开发的学习过程，包括：
- C# 脚本基础与游戏对象生命周期
- 物理、动画、UI 系统、事件系统
- 场景管理、资源管理（AssetBundle、Addressables）
- 小型 Demo 或练习项目的技术拆解与记录

---

## 📌 仓库定位

本仓库旨在作为个人长期成长的技术知识库，内容将持续更新，覆盖：

- 学习笔记  
- 项目经验  
- 技术设计文档  
- 各类踩坑记录  
- 思维框架与复盘

## 🔧 使用方式

- 直接在各对应目录中阅读 Markdown 文档。
- 文档均以独立主题组织，可作为学习路线的一部分。
- 如文档内容涉及示例代码，请配合仓库中其他代码文件一起查看。
